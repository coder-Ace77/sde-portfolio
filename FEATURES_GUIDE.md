# Quick Guide: How to Use New Features

## ðŸ“˜ Enhanced Callouts

Use these special callout blocks in your markdown notes:

### Note (Blue)
```markdown
> [!NOTE]
> This is an informational note with a blue theme and info icon.
```

### Info (Cyan)
```markdown
> [!INFO]
> General information appears in cyan with an alert icon.
```

### Warning (Amber)
```markdown
> [!WARNING]
> Caution messages appear in amber/yellow with a warning icon.
```

### Tip (Emerald Green)
```markdown
> [!TIP]
> Helpful tips appear in emerald green with a lightbulb icon.
```

### Success (Green)
```markdown
> [!SUCCESS]
> Success messages appear in green with a checkmark icon.
```

### Danger (Red)
```markdown
> [!DANGER]
> Critical warnings appear in red with a lightning icon.
```
