## Math class

Math class provides lots of utilities for handling the dsa problems.

```java
Math.max();
Math.min();
Math.abs();
Math.pow();
Math.sqrt();
double random = Math.random(); // [0,1]
Math.floor()
Math.round()
Math.ceil()

```
